解除重大ZX灾害（暴雨）II级应急响应

中国ZX局应急办                  2011年6月13日08时


〔总体情况〕

★自6月13日起，长江中下游等地将有一次强降水天气过程。其中，江汉南部、江南北部以及西南地区东部等地有大到暴雨，部分地区大暴雨，其间还将伴有短时雷雨大风等强对流天气。此次暴雨持续时间长、范围广、强度大，加之前期干旱造成土质较松，易发生山洪、山体滑坡、泥石流等灾害。

农业部 12日，农业部再次下发紧急通知，对防汛抗灾减灾工作进行全面部署，要求有关省（区）农业部门认真贯彻落实国家防总部署安排，把防御强降雨作为当前的紧迫任务，切实增强责任意识，狠抓措施落实，努力减轻灾害影响。
